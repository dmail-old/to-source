Object.toSource
=============

Object.prototype.toSource polyfill native firefox [Object.toSource](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/toSource).

## Example

```javascript
var array = [
	{name: 'foo'},
	{name: 'bar'}
];

array.toSource();
// '[{"name": "foo"}, {"name": "bar"}]'
```
